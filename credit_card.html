credit_card Module¶

Generate credit card random data.

forgery_py.forgery.credit_card.type()[source]¶

Return a random credit card type.

forgery_py.forgery.credit_card.check_digit(num)[source]¶

Return a check digit of the given credit card number.

Check digit calculated using Luhn algorithm (“modulus 10”) See: http://www.darkcoding.net/credit-card/luhn-formula/

forgery_py.forgery.credit_card.number(type=None, length=None, prefixes=None)[source]¶

Return a random credit card number.

Parameters:
  • type – credit card type. Defaults to a random selection.
  • length – length of the credit card number. Defaults to the length for the selected card type.
  • prefixes – allowed prefixes for the card number. Defaults to prefixes for the selected card type.
Returns:

credit card randomly generated number (int)
